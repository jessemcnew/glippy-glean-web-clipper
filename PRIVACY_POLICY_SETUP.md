## üöÄ Quick Setup (GitHub Pages)

### Step 1: Commit and Push Files

```bash
cd /Users/jmcnew/glippy
git add docs/
git commit -m "Add privacy policy HTML for Chrome Store hosting"
git push origin main
```

### Step 2: Enable GitHub Pages

1. Go to: https://github.com/jessemcnew/glippy-glean-web-clipper/settings/pages
2. Under **Source**, select:
   - **Branch**: `main`
   - **Folder**: `/docs`
3. Click **Save**
4. Wait 1-2 minutes

### Step 3: Get Your URL

Your privacy policy will be live at:
```
https://jessemcnew.github.io/glippy-glean-web-clipper/privacy-policy.html
```

### Step 4: Verify

1. Open the URL in your browser
2. Verify it displays correctly
3. Check it's publicly accessible

### Step 5: Use in Chrome Store

When submitting to Chrome Web Store, use this URL:
```
https://jessemcnew.github.io/glippy-glean-web-clipper/privacy-policy.html
```
