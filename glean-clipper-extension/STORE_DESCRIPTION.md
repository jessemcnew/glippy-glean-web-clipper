### How It Works

1. **Configure**: Set up your Glean API token and select a collection
2. **Clip**: Select text or right-click to clip content
3. **Sync**: Clips automatically sync to your Glean Collection
4. **View**: Open the notebook viewer to search and manage all your clips

### Privacy & Security

- All clips are stored locally in your browser
- API tokens are encrypted in Chrome's secure storage
- Only sends data to your configured Glean instance
- No third-party tracking or data collection
